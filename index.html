This project presents interactive visualizations of various datasets. It includes both interactive and static visualizations such as bar charts, heatmaps, and geographic maps to better understand the analyzed data. This example serves only as an illustration. The datasets were retrieved from Kaggle as noted on the referenced page.

In doing so, the data is extracted, loaded, transformed, and brought into the desired format using various methodologies of exploratory data analysis and data wrangling. Finally, they are visualized to support decision-makers with business metrics and to provide a foundation for future decisions.

The displayed visualizations were created with Python and R. Each section is labeled accordingly.

This is an ongoing personal project intended for practice and does not represent a final form.
